# print-rust-info

Displays the current version of the main **Rust** commands.

## Example

```yaml
steps:
  - uses: giancosta86/aurora-github/actions/print-rust-info@v2
```

## Inputs

|  Name   |    Type    |          Description           | Default value |
| :-----: | :--------: | :----------------------------: | :-----------: |
| `shell` | **string** | The shell used to run commands |   **bash**    |
